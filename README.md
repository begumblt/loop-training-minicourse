# Microsoft Loop Training Minicourse

A self-paced training course for Microsoft Loop, built as a single HTML file so any team can run it without an LMS, an admin, or a licence.

**[Start the course →](https://begumblt.github.io/loop-training-minicourse/)**

## Why

Workplace tools get adopted faster than they get taught. Standard enablement means a platform, a budget line, and someone to administer it — so for a tool like Loop it usually just doesn't happen, and people are handed a link and left to work it out.

This is the smallest thing that solves that: one file, no setup, no dependencies.

## How to use it

1. Open the course in a browser tab, or pin it as a tab in a Teams chat.
2. Open Microsoft Loop alongside it.
3. Work through the tutorial at your own pace, trying each step in Loop as you go.
4. Get a completion certificate at the end.
